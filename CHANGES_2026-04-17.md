## 1. Network Visualization: Node Size Fix

### Problem
Network plots had **oversized nodes** that overlapped and made the graph hard to read.

### Solution
Modified `src/visualization.py`:

```python
# Before
node_sizes = [degrees[node] * 100 for node in G.nodes()]  # Uncapped, too large

# After
base_sizes = [degrees[node] * 30 for node in G.nodes()]  # Reduced multiplier
node_sizes = [min(size, 800) for size in base_sizes]     # Cap max at 800
node_sizes = [max(size, 150) for size in node_sizes]     # Min 150 for readability
```

**Additional improvements:**
- Smaller font size (10pt → 9pt)
- Thinner colorbar (15px → 12px)

### Impact
- ✅ Cleaner network visualization

## 3. DeepSeek API Support

### Problem
DeepSeek API was geo-blocked from Alibaba Cloud servers, and there was no easy way to switch providers.

### Solution
Added **DeepSeek API support** with OpenAI-compatible interface:

#### Modified Files
1. `src/enhanced_keyword_extractor.py` - Hot topic analysis
2. `src/virtual_cell_tracker.py` - Paper summarization
3. `config.virtualcell.yaml` - Configuration template

#### Configuration
```yaml
llm:
  enabled: false  # Set to true to enable
  provider: deepseek  # Options: 'openai' or 'deepseek'
  model: deepseek-chat  # Options: 'gpt-4o-mini' or 'deepseek-chat'
  api_key: YOUR_API_KEY_HERE  # 🔑 PASTE YOUR KEY HERE
  base_url: https://api.deepseek.com
```

#### Key Features
- **OpenAI-compatible**: Uses same `openai` Python package
- **Easy switching**: Just change `provider` field
- **Placeholder key**: `YOUR_API_KEY_HERE` prevents accidental API calls
- **Both features**: Hot topic analysis + paper summarization

### How to Enable

1. **Get DeepSeek API key**: https://platform.deepseek.com/api-keys
2. **Edit config**: `config.virtualcell.yaml`
3. **Paste your key**: Replace `YOUR_API_KEY_HERE`
4. **Set enabled**: Change `enabled: false` → `enabled: true`
5. **Run pipeline**: `python run_virtualcell.py`

### Benefits vs OpenAI

| Feature | DeepSeek | OpenAI (GPT-4o-mini) |
|---------|----------|---------------------|
| **Cost** | ~$0.14/1M tokens | ~$0.15-0.60/1M tokens |
| **Quality** | Good for technical text | Excellent |
| **Speed** | Fast | Fast |
| **Geo-restrictions** | Fewer | More (blocks some regions) |
| **Context window** | 128K | 128K |

### Impact
- ✅ Cost-effective alternative to OpenAI
- ✅ Works from more geographic regions
- ✅ Easy to switch between providers
- ✅ Same codebase, just config change
